Shortcuts

While not an exhuaustive list, this shows the most common shortcuts available in Material Maker.

On macOS, Ctrl commonly corresponds to Cmd whereas equivalent of Alt is Option.

General

F1

Open user manual

Ctrl/Cmd-F1

Open user manual for selected library item

Ctrl/Cmd-,

Open preferences

Ctrl/Cmd-Q

Quit

Ctrl/Cmd-Shift-Q

Close project

Ctrl/Cmd-N

New material project

Ctrl/Cmd-S

Save project

Ctrl/Cmd-Shift-S

Save project as

Ctrl/Cmd-O

Load project

Ctrl/Cmd-Shift-N

New paint project

Ctrl/Cmd-Shift-E

Quick export

Ctrl/Cmd-E

Export material again

Graph Editor

MMB | Shift-LMB

Pan

G

Toggle grab mode on selected node(s)

Wheel

Zoom in/out

Wheel

Cycle between tabs (when hovering over tab bar)

Ctrl-MMB

Zoom view

Ctrl/Cmd-0

Reset zoom

Ctrl/Cmd

Snap nodes to grid (during node offsets)

Shift-LMB

Show selected node/port output in Preview2D(2)

LMB

Show selected node/port output in Preview2D

RMB | Space

Add node menu

RMB

Toggle node preview for hovered port

Alt-LMB

Lasso selection

LMB

Box selection

Ctrl-RMB

Connection knife tool

Ctrl/Cmd-Shift-RMB

Insert aperture pair on connection line

Shift-RMB

Insert reroute on connection line

X | Del | Backspace

Delete selected nodes

Ctrl/Cmd-D

Duplicate selected nodes

Ctrl/Cmd-Shift-D

Duplicate selected nodes with inuputs

F2 | Enter | LMB Double-click

Edit selected aperture node

Ctrl/Cmd-G

Create group from selected nodes

Alt-S

Swap inputs on selected node

Ctrl/Cmd-Shift-F

Frame selected node(s)

F

Color selected comment/aperture node(s)

C

Centers view

H

Minimize selected node(s)

Left | Right | Up | Down

Offsets view

Ctrl/Cmd-W

Make selected node(s) editable

Ctrl/Cmd-A

Select all

Ctrl/Cmd-Shift-A

Select none

LMB Double-click

Enters subgraph/nodegroup, or exit/go up a level in subgraph hierarchy when double-clicking in an empty area.

Ctrl/Cmd-L

Select source nodes from current selection

Ctrl/Cmd-Shift-L

Select target nodes fom current selection

Ctrl/Cmd-[

Align node(s) horizontally (start)

Ctrl/Cmd-\

Align node(s) horizontally (center)

Ctrl/Cmd-]

Align node(s) horizontally (end)

Ctrl/Cmd-I

Invert selection

Ctrl/Cmd-X

Cut selection

Ctrl/Cmd-C

Copy selection

Ctrl/Cmd-V

Paste selection

Ctrl/Cmd-Z

Undo

Ctrl/Cmd-Shift-Z

Redo

Nodes

Alt-LMB

Fine value adjustment

RMB

Open expression editor (float parameter)

Ctrl/Cmd-Wheel

Adjust field value incrementally. On hovered menus with a pop-up list of options, this cycles between the menu items.

Pan

Wheel

Zoom in/out

Ctrl/Cmd-LMB

Zoom view

MMB | Shift-LMB

Pan

Wheel

Zoom in/out

Ctrl/Cmd

Snap gizmos to grid

Shift

Snap gizmos to X/Y axis

Ctrl/Cmd-R

Center view

Ctrl/Cmd-LMB

Zoom view

Preview 3D

MMB | Alt-LMB

Orbit

Wheel

Zoom in/out

Ctrl/Cmd-Wheel

Adjust field of view

Shift-MMB

Offset view pivot

Shift-RMB

Rotate environment

RMB

3D preview options
