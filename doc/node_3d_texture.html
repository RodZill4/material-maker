3D texturesÂ¶

3D textures can be applied to objects defined by a heightmap, that can be generated either as a regular greyscale image, or from a 3D scene defined using signed distance functions.

In both cases, the Apply node is used to define 3D coordinates to query 3D textures, and the Select node can be used to apply different 3D textures depending on a color index.
