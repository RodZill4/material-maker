Quantize node
~~~~~~~~~~~~~

The **Quantize** node approximates the colors of the input 3D texture.

.. image:: images/node_3d_texture_quantize.png
	:align: center

Inputs
++++++

The **Quantize** node accepts one or more 3D textures.

Outputs
+++++++

The **Quantize** node outputs 3D textures.

Parameters
++++++++++

The **Quantize** node accepts the *number of steps* as parameter.
