Combine node
~~~~~~~~~~~~

The **Combine** node combines three grayscale 3D texture inputs into a 3D texture.

.. image:: images/node_3d_texture_combine.png
	:align: center

Inputs
++++++

The **Combine** node has 3 inputs, one for each (red, green, blue) channel.

Outputs
+++++++

The **Combine** node outputs a 3D texture.
