Uniform node
~~~~~~~~~~~~

The **Uniform** and **Uniform/Grayscale** node outputs a single uniform 3D texture
whose color is configurable.

.. image:: images/node_3d_texture_uniform.png
	:align: center

.. image:: images/node_3d_texture_uniform_grayscale.png
	:align: center

Inputs
++++++

The uniform node does not accept any input.

Outputs
+++++++

The uniform node provides an output that generates a uniform 3D color texture.

Parameters
++++++++++

The **Uniform** node has a single *color* parameter that defines the color of
the output 3D texture. The **Uniform/Grayscale** node has a single *float* parameter.
