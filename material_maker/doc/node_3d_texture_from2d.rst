From2D node
~~~~~~~~~~~

The **From2D** node generates a 3D texture from a 2D texture by projecting it along a specified axis.

.. image:: images/node_3d_texture_from2d.png
	:align: center

Inputs
......

The **From2D** node accepts a single 2D texture as input

Outputs
.......

The **From2D** node generates a 3D texture.

Parameters
..........

The **From2D** node accepts the projection *Axis* as a parameter.
