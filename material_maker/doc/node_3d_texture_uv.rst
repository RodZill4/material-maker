UV node
~~~~~~~

The **UV** node shows the 3D coordinates as a 3D texture.

.. image:: images/node_3d_texture_uv.png
	:align: center

Inputs
++++++

The **UV** node does not accept any input

Outputs
+++++++

The **UV** node outputs a single 3D texture.

Parameters
++++++++++

The **UV** node does not have any parameters.
